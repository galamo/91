# Create-react-app

Starter kit, tool

- https://create-react-app.dev/docs/getting-started
- `npm install -g create-react-app`
  Create Project name: my-app
- `npx create-react-app my-app`

- Navigate to the project
- `cd <Project/Folder Name>`
- `npm start`
